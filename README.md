## Set Bat

`java -cp java-decompiler.jar org.jetbrains.java.decompiler.main.decompiler.ConsoleDecompiler -dgs=true %1 %2`

`1.bat xxxx.jar decompiled`

decompiled 目录得到一个jar，然后修改后缀为zip等格式，解压缩得到java文件源代码。
